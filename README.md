# SpendBuddy
Developed a personal finance application that helps users track spending and manage expenses. The project focuses on user input handling, data organization, and basic financial calculations while strengthening Python programming and problem-solving skills.
Features
Record expenses
View spending information
Organize financial records
Simple command-line interface
Installation Instructions
Download the project files.
Ensure Python is installed on your computer.
Open a terminal in the project folder.
Run the application using Python.
Usage Instructions

Launch the program and follow the prompts to enter and manage expense information.
